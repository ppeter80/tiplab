Tiplab — nastavenie databázy

Postup: skopíruj Časť 1 → v Supabase SQL Editore Run → potom Časť 2 → Run.

① Časť 1 — tabuľky + dáta

  1. Supabase → SQL Editor → New query
  2. Vlož (Ctrl+V) → Run → má napísať „Success"

② Časť 2 — funkcie

  1. V editore New query (nový, prázdny)
  2. Vlož → Run → „Success"
